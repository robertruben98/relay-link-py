# relay-link-py

Python client for the Relay Protocol API (relay.link) — fast crosschain bridge/swap across 70+ chains. Sync + async, Pydantic v2.

## Install

```bash
pip install relay-link-py
```

The distribution is `relay-link-py`; the import package is `relay_link`.

## Quick start

```python
import relay_link
```

See the [API reference](api.md) for the full surface.
